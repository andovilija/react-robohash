## Install
```sh
npm install react-robohash --save
# or
yarn add react-robohash
```

## Usage
```
import Robohash from 'react-robohash';

<Robohash
  name="react-robohash"
  fileType="png|svg|jpg"
  set={1|2|3|4}
  bgset={1|2|3|4}
  gravatar={true|false}
  size={200|200x200}
/>
```
